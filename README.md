# Projetos-python

Este exercício foi desenvolvido com base no curso de Python Módulo 1, ministrado pelo professor Gustavo Guanabara, e tem como objetivo aplicar na prática os conceitos iniciais da linguagem. Ao longo do módulo, são apresentados fundamentos essenciais como entrada e saída de dados, tipos primitivos, variáveis, operadores e noções básicas de lógica de programação.

A realização do exercício contribui para a fixação desses conceitos, permitindo compreender como o Python funciona na prática e como estruturar um raciocínio lógico para a resolução de problemas simples. Além disso, o formato didático do curso facilita o aprendizado progressivo, tornando o conteúdo acessível mesmo para quem não possui experiência prévia em programação.

De modo geral, o exercício cumpre seu papel introdutório, reforçando a importância da prática constante no processo de aprendizagem e servindo como base para o aprofundamento em módulos mais avançados da linguagem Python.
